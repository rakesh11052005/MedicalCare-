# MedicalCare+

MedicalCare+ is an **AI-powered clinical decision support system**
designed to assist healthcare professionals in analyzing medical images,
starting with **pneumonia detection from chest X-ray images**.

The system is built with a strong focus on **patient safety**, **explainability**,
**ethical AI**, and **real-world industrial standards**.
