## 1. Model Overview

**Model Name:** Pneumonia Detection Model  
**Project:** MedicalCare+  
**Version:** v1.0.0  
**Model Type:** Binary Classification (Normal vs Pneumonia)  
**Imaging Modality:** Chest X-ray (CXR)

This model is designed to assist healthcare professionals by analyzing chest
X-ray images and providing probability-based indications of pneumonia.

---

## 2. Intended Use

### Primary Intended Use
- Clinical decision support for radiologists and physicians
- Assisting in screening and prioritization of chest X-ray cases
- Highlighting regions of interest using explainable AI techniques

### Intended Users
- Qualified healthcare professionals
- Radiologists
- Medical researchers (non-clinical use)

### Out-of-Scope Use
- Standalone medical diagnosis
- Direct patient-facing use
- Treatment decision-making without clinician oversight
